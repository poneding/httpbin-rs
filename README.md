# httpbin-rs

使用 Rust 实现 [httpbin](https://httpbin.org)。

## 目标

- 使用 Rust 及其生态，学习并记录；
- 实现 httpbin 的所有功能；

## 技术栈

- [rust](https://www.rust-lang.org/)
- [actix-web](https://actix.rs/)
